## Install

Install project dependencies

    $ npm install

## Setup

Set environment

    $ export NODE_ENV=development

## Server

Start server

    $ node server.js

## Serve

    $ npm run start

## Technologies

* React (CRA)
* Node (Express)
* HTML5 / CSS3 (BEM) 
* Webpack
